# Circle Matching App

## 📌 プロジェクト概要
音楽サークルやコミュニティ向けのマッチングアプリ。  
ユーザーは1つのアカウントで複数のサークルに参加でき、  
サークルごとに表示名を変えることも可能。


- 
## 🧩 設計メモ

### ■ アカウント仕様
- ユーザーアカウントは 1 つで複数サークルに参加可能
- サークルごとに「サークル内表示名」を設定できる
- サークル内表示名が未入力の場合は共通ユーザー名を使用する

### ■ UI / フロントエンド方針
- まずは画面の骨組み（Signup / Login / Register など）を作成
- 画面遷移とフォーム入力が動く状態を優先
- 見た目（デザイン）は後回しで OK
- API 接続は UI が揃ってから順次実装する

### ■ バックエンド方針（FastAPI）
- `/signup` `/login` の API を先に作成し、フロントと接続
- DB（SQLite）を後から接続し、ユーザー情報を保存
- パスワードはハッシュ化して保存する
- 認証は JWT を使用予定

### ■ 今後の実装予定
- サークル登録 API
- サークル参加 API
- サークル一覧 / 詳細取得 API
- 認証状態の管理（フロント側で JWT を保持）
